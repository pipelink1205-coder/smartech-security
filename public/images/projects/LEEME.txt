# Galería de trabajos (mapa + evidencias)

## 1. Registrar en MySQL (tabla `projects`)

Campos importantes para el mapa:

| Campo | Ejemplo |
|-------|---------|
| category | Cámaras de Seguridad 4K |
| description | Texto de la tarjeta |
| address | Carrera 72 # 11-11, Medellín |
| latitude | 6.24915 |
| longitude | -75.59085 |
| comuna_numero | 10 (solo comunas de Medellín, 1-16) |
| location | La Candelaria (referencia) |

Fuera de Medellín (Envigado, Bello, Itagüí): deja comuna_numero en NULL;
el pin igual aparece con latitude/longitude.

## 2. Fotos evidencia (tabla `project_images`)

path: images/projects/mi-trabajo/01.jpg
project_id: id del proyecto
sort_order: 0, 1, 2...

## 3. Archivos en disco

public/images/projects/mi-trabajo/01.jpg

- Comunas con trabajos = verde más intenso
